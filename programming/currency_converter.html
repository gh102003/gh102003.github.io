Currency Converter

Currency Converter is a project that was worked on for the whole of 2017. It started as a simple application that could convert currencies, but its feature set grew over time. This includes:

  • updating currencies from the internet
  • a 'Currency Manager' where you can create custom currencies
  • a saves feature for remembering conversions

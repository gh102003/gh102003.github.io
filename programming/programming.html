Programming

I know multiple programming languages, although I'm not very experienced with using all of them.

Java

I'm most familiar with using Java because this is what Minecraft was made in. When I began building desktop applications, Java was a natural starting point because I had already used it when I was experimenting with Forge mods.

I'd like to experiment more with the streams functionality in Java, and also get into the JavaFX library. I've already played with various JSON libraries, including json.simple and GSON.

My Currency Converter was first application I ever made with Swing. I used it as an opportunity to learn about saving files, serialising JSON and using libraries.

Python

I think that Python is a great language for small data projects. I've used it for creating simple neural networks as well as making games like Breakout. I started using it on a Raspberry Pi, where it was the recommended language for using the GPIO pins.

The list comprehension and slice syntax is very intuitive in Python, so I have recently been experimenting with them.

C#

I've experimented with C# for Universal Windows Platform apps (like PhotoTweak), and for Unity scripts. I'm not very experienced with it, although I plan to look into developing games and native win32 applications with it.

Javascript

I used node.js for the server side of my Noughts and Crosses game and jQuery for the client side. jQuery's selectors and AJAX shortcuts made my project much more streamlined.

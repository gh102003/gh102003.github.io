Flash

Flash is a simple flashcards web app. Although it's still in development, there is an early version you can try. Please note that this isn't always live, so if it doesn't work, please try again later.

Flash has been an insight for me into React.js. It also uses React Router and a few other smaller libraries.

Features:

  • Categories — can be dragged into other categories
  • Colour coding — categories can be coloured, either from the palette or a hex code
  • Full screen mode — to view a flashcard isolated from the rest
  • Quizzes — to test your knowledge

Later on, I will add a dark theme and I will optimise the loading of categories. I may decide to add an authentication system, so only users with permission can edit the cards; this would make Flash more suitable for real-world use.

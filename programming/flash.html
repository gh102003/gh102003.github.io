Flash

Flash is a simple flashcards web app. Although it is still in development, you can find an early version here. Please note that this may not always be live, so if it doesn't work, please try again later.

Flash has been an insight for me into React.js. It also uses React Router and a few other smaller libraries.

Features:

  • Categories
  • Colour coding
  • Full screen mode
  • Quizzes
